# 🚀 AI Career Guidance System

An intelligent AI-powered career guidance platform built using **Python**, **Streamlit**, and **Google Gemini AI**.

This project helps students and job seekers with:
- Career Guidance
- AI Resume Analysis
- Theory Tests
- Coding Tests
- Interview Preparation
- Professional Profile Management

---

# 🌟 Features

## 📌 Career Guidance
- Personalized career suggestions
- Skill recommendations
- AI-generated learning roadmap

## 📄 Resume Checker
- Upload resume (PDF)
- AI analyzes resume
- Finds:
  - Matching skills
  - Missing skills
  - Improvement suggestions

## 💻 Coding Test
- AI-generated coding questions
- Code editor section
- Code analysis support

## 🧠 Theory Test
- AI-generated MCQ/theory questions
- Skill evaluation

## 🎤 Interview Preparation
- Technical interview questions
- HR interview preparation

## 👤 Professional Profile
- Editable professional profile
- Avatar selection
- Skills and certifications
- Career information storage

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/surya70104/ai-career-guidance-system.git
```

## 2️⃣ Move Into Project

```bash
cd ai-career-guidance-system
```

## 3️⃣ Install Requirements

```bash
pip install -r requirements.txt
```

## 4️⃣ Run Application

```bash
streamlit run app.py
```
